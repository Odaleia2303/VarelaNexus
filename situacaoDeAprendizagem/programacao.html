Lógica de Programação

1. O que são Linguagens de Programação?

São conjuntos de regras usadas para criar programas. Servem para dar instruções ao computador.

C++: Compilada, muito rápida, usada em sistemas e jogos.

JavaScript: Interpretada, usada na web.

Java: Orientada a objetos, usada em empresas.

Python: Interpretada, simples e usada em IA.

PHP: Muito usada em back-end web.

2. Interpretada x Compilada

Interpretada: executada linha por linha (JS, Python, PHP).

Compilada: convertida para código de máquina antes (C++, Java).

3. O que é HTML e CSS?

HTML: Linguagem usada para estruturar páginas web.

CSS: Linguagem usada para estilizar páginas web.

4. O que é uma IDE?

IDE é um ambiente de desenvolvimento com ferramentas para programar.

Exemplos: VSCode, IntelliJ, NetBeans.
