Fundamentos de Redes

1. O que são Redes de Computadores?

São estruturas que conectam dispositivos para trocar informações, permitindo comunicação, compartilhamento de arquivos, internet e serviços.

2. O que é uma Rede Mínima?

É o menor tipo de rede possível: apenas dois computadores conectados entre si.

3. O que é a Internet? O que compõe?

A Internet é a maior rede do mundo.
Ela é composta por:
• Cabos de fibra óptica (submarinos e terrestres)
• Roteadores e switches
• Provedores de internet (ISP)
• Servidores
• Satélites e radiofrequência
• Protocolos de comunicação (TCP/IP)

4. O que é Web?

É o conjunto de sites, páginas e serviços acessados via navegador usando HTTP/HTTPS.

5. Mapa Global de Fibra Óptica

A fibra óptica conecta continentes e permite internet em alta velocidade.

6. Componentes de Hardware de uma Rede Wi-Fi

• Roteador Wi-Fi
• Antenas
• Modem
• Repetidores
• Placas Wi-Fi (PCs e celulares)

7. Hardware de Rede com Cabo

• Switch
• Roteador
• Modem
• Cabos Ethernet (Cat5, Cat6)
• Placas de Rede

8. O que é Protocolo de Comunicação?

Conjunto de regras que definem como os dados são transmitidos na rede.

9. Modelo OSI

Modelo teórico com 7 camadas:
1 - Física
2 - Enlace
3 - Rede
4 - Transporte
5 - Sessão
6 - Apresentação
7 - Aplicação

10. Modelo TCP/IP

Modelo usado na Internet:
• Aplicação
• Transporte
• Rede
• Enlace

11. IPv4 e IPv6

IPv4: 32 bits — Ex.: 192.168.0.1
IPv6: 128 bits — Ex.: 2001:0db8:85a3::8a2e

12. Configuração de IP e TCP em Node.js

const net = require('net');
const server = net.createServer();
server.listen(3000, '127.0.0.1');
console.log('Servidor ativo');

13. O que é HTTP?

Protocolo usado para transferir páginas e dados na Web.

14. O que é HTTPS?

É a versão segura do HTTP, usando criptografia SSL/TLS.

15. Framework HTTP em Node.js

Express.js — usado para criar APIs e servidores Web.

16. Exemplos Backend (Node.js)

// 1 - Enviar texto
app.get('/msg', (req, res) => {
  res.send('Olá Front!');
});

// 2 - Receber dados do front
app.post('/dados', (req, res) => {
  console.log(req.body);
  res.send('Dados recebidos');
});

// 3 - Enviar JSON
app.get('/json', (req, res) => {
  res.json({ status: 'ok', mensagem: 'Teste' });
});
